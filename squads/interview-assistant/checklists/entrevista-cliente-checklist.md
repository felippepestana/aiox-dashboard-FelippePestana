# Checklist: Entrevista com Cliente

## 1. Preparação
- [ ] Área jurídica definida
- [ ] Dados do cliente pré-carregados (se existente)
- [ ] Microfone testado e funcionando
- [ ] Serviço de transcrição disponível
- [ ] Banco de perguntas da área carregado

## 2. Abertura da Entrevista
- [ ] Cliente identificado e qualificado
- [ ] Área jurídica confirmada
- [ ] Gravação iniciada
- [ ] Transcrição em tempo real ativa
- [ ] Painel de sugestões visível

## 3. Coleta de Informações Essenciais
- [ ] Fatos narrados pelo cliente
- [ ] Datas-chave identificadas
- [ ] Partes envolvidas nomeadas
- [ ] Valores/prejuízos quantificados
- [ ] Documentos disponíveis mapeados
- [ ] Testemunhas identificadas
- [ ] Tentativas prévias de resolução documentadas

## 4. Informações por Área (conforme aplicável)

### 4.1 Trabalhista
- [ ] Data de admissão
- [ ] Data e tipo de demissão
- [ ] Salário atual/último
- [ ] Jornada de trabalho
- [ ] Existência de horas extras
- [ ] FGTS e férias
- [ ] Condições de trabalho (insalubridade/periculosidade)

### 4.2 Civil
- [ ] Descrição detalhada do dano
- [ ] Relação contratual documentada
- [ ] Provas do prejuízo
- [ ] Notificação extrajudicial prévia
- [ ] Prescrição verificada

### 4.3 Consumidor
- [ ] Produto/serviço identificado
- [ ] Nota fiscal/comprovante
- [ ] Descrição do defeito/problema
- [ ] Contato prévio com fornecedor
- [ ] Registro no Procon

### 4.4 Família
- [ ] Existência de filhos menores
- [ ] Regime de bens
- [ ] Patrimônio a partilhar
- [ ] Necessidade de alimentos
- [ ] Medidas protetivas em vigor

## 5. Sugestões de IA
- [ ] Perguntas sugeridas revisadas
- [ ] Precedentes relevantes anotados
- [ ] Alertas de informação faltante resolvidos
- [ ] Dicas estratégicas consideradas

## 6. Encerramento
- [ ] Gravação finalizada
- [ ] Transcript revisado para trechos de baixa confiança
- [ ] Pontos importantes marcados
- [ ] Notas do advogado adicionadas
- [ ] Próximos passos comunicados ao cliente

## 7. Pós-Entrevista
- [ ] Estratégia gerada e revisada
- [ ] Teses jurídicas validadas
- [ ] Provas faltantes listadas para coleta
- [ ] Esboço de peça gerado (se aplicável)
- [ ] Relatório de entrevista exportado
- [ ] Agendamento de retorno definido
