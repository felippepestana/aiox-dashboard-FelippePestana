# Checklist: Fechamento Financeiro Mensal

## 1. Honorários e Faturamento
- [ ] Notas fiscais do mês emitidas
- [ ] Faturamento previsto vs. realizado conferido
- [ ] Recebíveis atualizados
- [ ] Aging de recebíveis atualizado
- [ ] Inadimplência identificada e ações propostas
- [ ] Honorários sucumbenciais registrados
- [ ] Alvarás de levantamento acompanhados
- [ ] Rateio entre sócios calculado

## 2. Tributos
- [ ] Faturamento bruto consolidado
- [ ] Retenções na fonte identificadas
- [ ] IRPJ apurado e guia gerada
- [ ] CSLL apurada e guia gerada
- [ ] PIS apurado e guia gerada
- [ ] COFINS apurada e guia gerada
- [ ] ISS apurado e guia gerada
- [ ] INSS sobre pró-labore calculado
- [ ] Obrigações acessórias verificadas (DCTF, EFD, ECD)
- [ ] Carga tributária efetiva calculada

## 3. Fluxo de Caixa
- [ ] Saldo bancário consolidado
- [ ] Entradas do mês registradas
- [ ] Saídas do mês registradas
- [ ] Projeção para próximos 3 meses elaborada
- [ ] Cenários (otimista, realista, pessimista) montados
- [ ] Gaps de caixa identificados
- [ ] Runway calculado

## 4. Provisões
- [ ] Processos passivos revisados
- [ ] Classificação de risco atualizada
- [ ] Provisões calculadas (prováveis)
- [ ] Ajustes de provisão registrados
- [ ] Relatório para notas explicativas preparado

## 5. Consolidação
- [ ] DRE do mês elaborado
- [ ] Indicadores financeiros calculados
- [ ] Comparativo com orçamento realizado
- [ ] Comparativo com mesmo período do ano anterior
- [ ] Desvios significativos explicados
- [ ] Alertas emitidos
- [ ] Recomendações formuladas
- [ ] Relatório gerencial entregue aos sócios
