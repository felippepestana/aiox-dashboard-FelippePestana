# Checklist: Análise Processual Completa

## 1. Recepção e Classificação
- [ ] PDF dos autos recebido e legível
- [ ] Tipo de demanda classificado
- [ ] Urgência avaliada
- [ ] Workflow definido
- [ ] Tarefas distribuídas

## 2. Extração de Dados
- [ ] Processo identificado (número CNJ, vara, comarca)
- [ ] Partes qualificadas (autor, réu, advogados)
- [ ] Pedidos da inicial mapeados
- [ ] Decisões interlocutórias identificadas
- [ ] Mandados expedidos catalogados
- [ ] Cronologia processual construída
- [ ] Prazos em curso calculados

## 3. Análise de Vícios
- [ ] Pressupostos processuais verificados
- [ ] Competência do juízo analisada
- [ ] Fundamentação das decisões examinada (art. 489, §1º)
- [ ] Contraditório prévio verificado (arts. 9º e 10)
- [ ] Regularidade das intimações conferida
- [ ] Proporcionalidade das medidas avaliada
- [ ] Vícios classificados (absolutos/relativos)
- [ ] Dispositivos violados indicados

## 4. Pesquisa Jurisprudencial
- [ ] Temas de pesquisa definidos
- [ ] STF pesquisado (súmulas, repercussão geral)
- [ ] STJ pesquisado (súmulas, repetitivos, jurisprudência)
- [ ] Tribunal estadual pesquisado
- [ ] Mínimo 5 precedentes selecionados
- [ ] Ementas completas com dados de citação
- [ ] Convergência/divergência identificada

## 5. Mapeamento Legislativo
- [ ] Dispositivos constitucionais mapeados
- [ ] Dispositivos do CPC identificados
- [ ] Legislação especial verificada
- [ ] Conformidade da decisão analisada
- [ ] Violações documentadas

## 6. Estratégia Processual
- [ ] Opções processuais mapeadas
- [ ] Probabilidade de êxito avaliada
- [ ] Peça principal recomendada
- [ ] Peças subsidiárias definidas
- [ ] Tese principal estruturada
- [ ] Argumentos subsidiários ordenados
- [ ] Riscos avaliados

## 7. Minutagem da Peça
- [ ] Template correto selecionado
- [ ] Endereçamento preenchido
- [ ] Partes qualificadas
- [ ] Fatos narrados
- [ ] Fundamentação jurídica desenvolvida
- [ ] Jurisprudência incorporada
- [ ] Pedidos redigidos (principal + subsidiários)
- [ ] Pedido de urgência incluído (se aplicável)

## 8. Revisão
- [ ] Aspectos formais verificados
- [ ] Fundamentação revisada
- [ ] Jurisprudência conferida
- [ ] Coerência argumentativa validada
- [ ] Pedidos completos e coerentes
- [ ] Linguagem revisada
- [ ] Classificação de qualidade atribuída (A/B/C/D)

## 9. Entrega
- [ ] Peça final consolidada
- [ ] Relatório de análise preparado
- [ ] Entregue ao usuário
