# Checklist: Peça Urgente

## 1. Triagem (5 min)
- [ ] Urgência confirmada como CRÍTICA
- [ ] Mandado identificado (tipo, status, objeto)
- [ ] Prazo remanescente calculado
- [ ] Workflow urgente acionado

## 2. Leitura Rápida (15 min)
- [ ] Decisão impugnada localizada e lida
- [ ] Mandado expedido identificado
- [ ] Partes qualificadas
- [ ] Data de intimação identificada
- [ ] Prazo de recurso calculado

## 3. Análise Rápida (10 min)
- [ ] Vício principal identificado
- [ ] Dispositivo legal violado indicado
- [ ] Fundamentação da decisão avaliada
- [ ] Gravidade do vício classificada

## 4. Jurisprudência Urgente (10 min)
- [ ] 3-5 precedentes fortes selecionados
- [ ] Ementas com dados completos
- [ ] Precedentes do tribunal competente priorizados

## 5. Estratégia (5 min)
- [ ] Peça mais eficaz definida
- [ ] Tese principal escolhida
- [ ] Pedido de suspensão formulado

## 6. Minutagem (20 min)
- [ ] Peça redigida com fundamentação mínima
- [ ] Pedido de urgência destacado
- [ ] Pedido de suspensão do mandado incluído
- [ ] Pedido de manutenção de posse incluído

## 7. Revisão Express (10 min)
- [ ] Endereçamento correto
- [ ] Pedidos completos
- [ ] Fundamentação mínima presente
- [ ] Sem erros formais graves

## 8. Entrega Imediata
- [ ] Peça pronta para protocolo
- [ ] Instruções de protocolo fornecidas
