# Minutar Peça Processual

## Objetivo
Elaborar a minuta da peça processual recomendada pelo estrategista, com fundamentação técnica completa e formatação adequada.

## Agente Responsável
`minutador-pecas`

## Inputs
- Parecer estratégico (`estrategista-juridico`)
- Jurisprudência selecionada (`pesquisador-jurisprudencial`)
- Mapeamento legislativo (`analista-legislativo-processual`)
- Dados dos autos (`leitor-processual`)

## Passos
1. Identificar o tipo de peça a ser elaborada
2. Selecionar o template adequado
3. Preencher dados de identificação (endereçamento, partes, processo)
4. Redigir a síntese dos fatos
5. Desenvolver a fundamentação jurídica (lei + jurisprudência)
6. Estruturar os argumentos conforme tese definida
7. Redigir os pedidos (principal, subsidiários, urgência)
8. Revisar formatação e completude
9. Enviar para revisão

## Critérios de Qualidade
- Endereçamento correto
- Fundamentação com base em pelo menos 3 dispositivos legais
- Pelo menos 3 precedentes jurisprudenciais citados
- Pedidos claros, específicos e determinados
- Pedido de urgência quando aplicável
- Linguagem técnica e objetiva

## Output
Minuta completa da peça processual pronta para revisão pelo `revisor-juridico`.
