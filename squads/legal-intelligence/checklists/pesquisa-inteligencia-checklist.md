# Checklist: Pesquisa de Inteligência Jurídica

## 1. Definição de Escopo
- [ ] Tema de pesquisa claramente definido
- [ ] Tribunais-alvo selecionados
- [ ] Período de pesquisa definido
- [ ] Contexto do caso fornecido (fatos, fundamentos, pedidos)
- [ ] Magistrado identificado (se aplicável)

## 2. Pesquisa de Precedentes
- [ ] STF pesquisado (súmulas vinculantes, repercussão geral)
- [ ] STJ pesquisado (súmulas, repetitivos, acórdãos)
- [ ] Tribunais estaduais/federais pesquisados
- [ ] Mínimo 10 precedentes selecionados
- [ ] Ementas completas com dados de citação
- [ ] Convergência/divergência mapeada
- [ ] Precedentes classificados por vinculância

## 3. Classificação Temática
- [ ] Assuntos CNJ mapeados
- [ ] Temas de repercussão geral identificados
- [ ] Status dos temas verificado (pendente/julgado/fixado)
- [ ] Teses fixadas transcritas
- [ ] Repetitivos do STJ verificados
- [ ] Impacto na estratégia avaliado

## 4. Análise de Similaridade
- [ ] Elementos comparáveis extraídos
- [ ] Similaridade fática calculada (30%)
- [ ] Similaridade jurídica calculada (40%)
- [ ] Similaridade de pretensão calculada (20%)
- [ ] Similaridade processual calculada (10%)
- [ ] Score global calculado
- [ ] Distinguishing identificados
- [ ] Ranking de precedentes gerado

## 5. Consolidação
- [ ] Todas as análises coletadas
- [ ] Cruzamento de dados realizado
- [ ] Probabilidade de êxito avaliada
- [ ] Pontos fortes e fracos identificados
- [ ] Recomendações estratégicas formuladas
- [ ] Relatório final gerado
