# Checklist: Peticionamento Eletrônico

## 1. Validação de Documentos
- [ ] PDF em formato PDF/A (1b ou 2b)
- [ ] Tamanho do arquivo dentro do limite do tribunal
- [ ] Resolução mínima de 200 DPI (se digitalizado)
- [ ] Documento legível e completo
- [ ] Anexos em formato aceito

## 2. Assinatura Digital
- [ ] Certificado digital válido (não expirado)
- [ ] Certificado ICP-Brasil (A1 ou A3)
- [ ] Assinatura aplicada corretamente ao PDF
- [ ] Cadeia de certificação verificada
- [ ] Timestamp de assinatura presente

## 3. Dados do Processo
- [ ] Número do processo conferido (formato CNJ)
- [ ] Tribunal e vara corretos
- [ ] Processo em andamento (não arquivado)
- [ ] Prazo processual ainda vigente
- [ ] Tipo de petição classificado corretamente

## 4. Envio
- [ ] Sistema-alvo identificado (PJE, e-SAJ, e-Proc)
- [ ] Credenciais de acesso válidas
- [ ] Petição principal enviada
- [ ] Anexos enviados (se houver)
- [ ] Número de recibo obtido
- [ ] Comprovante de protocolo gerado

## 5. Confirmação
- [ ] Protocolo confirmado no sistema judicial
- [ ] Comprovante armazenado
- [ ] Solicitante notificado
- [ ] Log de peticionamento atualizado
