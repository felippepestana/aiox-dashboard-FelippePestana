# Checklist: Monitoramento Diário

## 1. Preparação
- [ ] Lista de processos monitorados carregada
- [ ] Lista de partes e advogados configurada
- [ ] Tribunais-alvo definidos
- [ ] Credenciais de API validadas
- [ ] Calendário de feriados atualizado

## 2. Varredura DJE
- [ ] DJE de todos os tribunais configurados consultados
- [ ] Busca por número de processo executada
- [ ] Busca por nome de parte executada
- [ ] Busca por OAB do advogado executada
- [ ] Publicações relevantes identificadas
- [ ] Publicações classificadas por tipo

## 3. Rastreamento de Movimentações
- [ ] APIs judiciais acessíveis
- [ ] Todos os processos da carteira consultados
- [ ] Novas movimentações identificadas
- [ ] Movimentações classificadas por código CNJ
- [ ] Relevância atribuída a cada movimentação
- [ ] Processos parados sinalizados

## 4. Sincronização de Prazos
- [ ] Prazos do DJE coletados
- [ ] Prazos das APIs coletados
- [ ] Duplicatas removidas
- [ ] Feriados e suspensões considerados
- [ ] Dies ad quem recalculado em dias úteis
- [ ] Prazos classificados por urgência
- [ ] Conflitos de agenda identificados

## 5. Alertas e Entrega
- [ ] Alertas de prazos críticos gerados
- [ ] Alertas de movimentações críticas gerados
- [ ] Relatório diário consolidado
- [ ] Notificações enviadas para responsáveis
- [ ] Log de execução registrado
