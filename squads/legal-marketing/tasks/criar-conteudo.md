# Criar Conteúdo Jurídico

## Objetivo
Produzir conteúdo jurídico informativo e educativo para as plataformas do escritório, otimizado para a plataforma-alvo e em conformidade com as regras da OAB.

## Agente Responsável
`criador-conteudo-juridico`

## Inputs
- Tema do conteúdo
- Plataforma-alvo (blog, LinkedIn, Instagram, newsletter)
- Área do direito
- Público-alvo (empresas, pessoas físicas, profissionais)
- Palavras-chave para SEO (se blog)

## Passos
1. Pesquisar tema e reunir informações jurídicas atualizadas
2. Identificar ângulo informativo adequado para o público-alvo
3. Estruturar conteúdo conforme formato da plataforma
4. Redigir conteúdo com linguagem acessível e juridicamente precisa
5. Incluir referências legais (artigos de lei, jurisprudência)
6. Otimizar para SEO (se blog: título, meta description, headings)
7. Adaptar para formato visual (se redes sociais: cards, carrosséis)
8. Verificar conformidade básica com regras OAB
9. Enviar para validação de compliance

## Output Estruturado
```yaml
conteudo:
  titulo: ""
  plataforma: ""
  area_direito: ""
  tipo: "" # artigo, post, newsletter, ebook
  formato: "" # texto, carrossel, video_roteiro, infografico
  corpo: ""
  referencias_legais: []
  palavras_chave: []
  meta_description: ""
  status: "pendente_compliance"
```

## Critérios de Qualidade
- Conteúdo juridicamente preciso e atualizado
- Linguagem acessível ao público-alvo
- Sem promessas de resultado ou linguagem mercantilista
- Otimizado para a plataforma-alvo
- Referências legais incluídas
